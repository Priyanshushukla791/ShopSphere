# ShopSphere – Personalized Shopping Website 🛒

## Overview

**ShopSphere** is a personalized shopping website inspired by modern e-commerce platforms like Amazon.
It provides a clean and user-friendly interface where users can browse different product categories such as electronics, furniture, clothing, toys, and more.

This project demonstrates the use of **HTML, CSS, and JavaScript** to build a responsive and interactive online shopping webpage.

---

## Features

* Modern **navigation bar with logo**
* **Search bar** for product searching
* Product category sections such as:

  * Bed Covers
  * Health & Personal Care
  * Furniture
  * Electronic Devices
  * Beauty Products
  * Pet Care
  * Toys
  * Clothes
* Interactive **shopping cart icon**
* Attractive **hero section with background image**
* Responsive **product grid layout**
* Footer section with useful links
* Simple **JavaScript interactions** like search alerts and cart click events

---

## Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
* **JavaScript** – Basic interactivity
* **Font Awesome** – Icons used in the navbar and cart

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any browser.
4. The ShopSphere shopping webpage will load.

---

## Purpose of the Project

The goal of this project is to practice **front-end web development skills** by creating a simple e-commerce interface similar to popular online shopping platforms.

It helps in understanding:

* Web page structure
* Responsive layouts
* UI design concepts
* Basic JavaScript interactions
